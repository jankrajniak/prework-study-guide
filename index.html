Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The body is structured into section using semantic elements

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • A CSS stylesheet must be associated via link command in the header, including the rel and href modifiers

Git

  • Git is used for version control
  • GitHub is a cloud-based repository used in tandem with Git for version control
  • GitHub provides remote access to all files, branches, issues associated with your project
  • The checkout a new branch, enter the command git checkout -b NAMEOFBRANCH. if the branch is existing, omit -b
